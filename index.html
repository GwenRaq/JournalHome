Learning how to structure a web application

I learned how to make an html website. I made the website with Sublime Text and Git Shell. First, I learned the git commands which are git add, git commit -m, git status, and git push. Git add means your adding a specific file to sublime text. Git commit means you're initializing the project. Git status is for checking your projects status and git push means that you're uploading everything into your respository.

Next, I learned some of the basics or components for putting a website together on sublime text. I leanrned how to make a paragraph element, a closing tag, a header, a block element, an inline element, how to add an image, and how to make a closing tag.
